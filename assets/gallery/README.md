# 갤러리 사진 추가하는 법 (모바일에서)

컴퓨터·클로드 없이, 휴대폰만으로 활동 사진을 추가할 수 있어요.

## 사진 추가 (1분)

1. 휴대폰 브라우저로 이 폴더 열기 →
   `https://github.com/Godeng/social-impact-portfolio/tree/main/assets/gallery`
   (한 번 북마크해두면 편해요)
2. **Add file → Upload files** 누르기
3. 사진 고르고 → 맨 아래 **Commit changes** 누르기
4. 끝. 1~2분 뒤 포트폴리오 페이지를 새로고침하면 사진이 떠요.

> 파일 이름은 아무거나 괜찮아요(IMG_1234.jpg 그대로 OK).
> 이름을 안 바꾸면 캡션이 파일 이름으로 나와요 — 나중에 아래에서 예쁘게 바꾸면 됩니다.

## 캡션 예쁘게 달기 (선택, 시간 날 때)

`captions.json` 파일을 열고(연필 아이콘 ✏️), 목록에 한 줄 추가:

```json
{ "file": "IMG_1234.jpg", "tag": "지역사회", "title": "활동 이름", "when": "2025 여름", "desc": "한 줄 설명" }
```

- `file` = 방금 올린 사진의 파일 이름 (똑같이)
- `tag` = 필터에 쓰이는 분류 (예: 지역사회 / 커뮤니티 / 교육봉사 — 새 태그를 써도 자동으로 필터가 생겨요)
- 쉼표(`,`)로 항목을 구분하고, 마지막 항목 뒤에는 쉼표를 붙이지 마세요.

## 미리 만들어둔 자리 채우기

`captions.json`에 이미 활동 7개가 자리로 들어가 있어요(색 카드로 표시됨).
그 자리에 사진을 넣고 싶으면, 사진 파일 이름을 그 항목의 `file` 이름과
똑같이(예: `uzbekistan.jpg`) 해서 업로드하면 색 카드가 그 사진으로 바뀝니다.
